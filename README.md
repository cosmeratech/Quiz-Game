# Quiz Game

A C++ console-based quiz application that tests knowledge on data structures and programming concepts.

## Features

- 5 multiple-choice questions covering:
  - Data structures (FIFO, queues, stacks)
  - C++ memory management
  - Python data types and mutability
- Interactive gameplay with immediate feedback
- Score tracking and pass/fail determination

## How to Play

1. Compile the program:
   ```bash
   g++ QuizGame.cpp -o QuizGame
   ```

2. Run the executable:
   ```bash
   ./QuizGame
   ```

3. Follow the on-screen instructions to start the quiz.

4. Answer questions by selecting options (a, b, c, or d).

5. View your score at the end. Pass if you score 3 or higher.

## Requirements

- C++ compiler (g++ or compatible)
- Standard C++ library support
